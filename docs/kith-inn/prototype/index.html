街坊味 · 老板侧经营大脑

这版原型把 PRD 的主链路压进一屏：自然语言记单、确认草稿、菜单排程、订单台账、按楼栋分拣。按钮和对话调用同一组后端动作，详情页只做确定性视图。

orders draft / confirmed / canceled 分清记单生命周期；确认才进经营口径。
service_slots date + occasion 锚定午晚餐次；确认订单或发布菜单时打开。
offerings 菜、套餐、服务都走同一个枢纽；菜单和采购不另建 dish 表。
fulfillments 按 order item 建履约，反范式 serviceDate / occasion 支撑分拣。

色彩体系（暖）

砖红 #d7462f 主色 · 蔬菜绿 #1f8a61（已付/已送）· 琥珀 #b96d12（待办/未付/未送）· 米白 #fffaf1。区别于 NutUI 默认 JD 冷红——贴"灶台 / 熟人 / 烟火气"，暖中性色、大字号大触控区（适老）。落地覆盖 --nutui-color-primary 等 CSS 变量。

NutUI 组件 / 图标映射

Tabbar底部 今天/菜单/订单/送餐，icon 用 @nutui/icons-react
Cell / Tag订单·菜单·楼栋行；午晚/主料/付款/自家标签
Button · Progress · Avatar · Input确认换菜、楼栋进度条、顾客/味头像、语音粘贴条

三版评审 · 取长补短

底 = 本版（暖色 · 4 tab · 状态交互 · 全屏 PRD 语义最深）；补 codex 版的"2 天窗口提示""分拣汇总 chips"。codex 版（tmp/）用 NutUI 默认 JD 冷红 #fa2c19，通用但不贴品牌；另有 terracotta 暖色互动版，色彩同向、内容较简。综合取本版为最合适。
