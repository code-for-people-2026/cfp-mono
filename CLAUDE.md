# cfp-mono — Claude / Agent 指令

> 本文件会被每个 Claude Code session 自动加载并当作指令遵守。团队级流程写在这里，确保所有人（及其 AI session）一致。

## PR 流程（重要）

本仓库的 PR 发出后会由 **Codex 自动 review**。合并前必须：

1. 逐条 review Codex 的每条 comment；
2. 判断是否真的需要改——**该改的改；不该改的，回复说明理由**（不无脑照改，也不无脑忽略）；
3. **所有重要 comment 都 resolve 掉**，PR 才能合并。

开 PR / push 属外发动作，需先与发起人确认。
